# Timeblocks

一个基于 Flutter 开发的高效时间块管理应用，帮助你直观地规划和统计每一分钟。

## ✨ 主要功能

*   **无限时间轴**：支持从 1925 年到未来的无限滚动，精确到 5 分钟粒度。
*   **拖拽规划**：长按并拖动即可快速选中时间范围进行涂色。
*   **双层级管理**：支持“事件类别”与“事件内容”两级分类，统计更精细。
*   **可视化统计**：内置条形图与饼图，支持今日、本周、自定义等多种时间范围筛选。
*   **高度定制**：内置 HSV 专业取色器，可为每个事件定义专属颜色。
*   **数据保存**：基于本地存储，重启应用数据不丢失。

## 🚀 运行项目

1.  **安装依赖**
    ```bash
    flutter pub get
    ```

2.  **运行应用**
    ```bash
    flutter run
    ```
